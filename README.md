House Price Prediction using Linear Regression

Overview
This project implements a Linear Regression model to predict house prices using Scikit-learn. It includes data preprocessing, model training, prediction, evaluation, and visualization.

Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

Project Files
- `house_price_prediction.ipynb` – Jupyter Notebook
- `dataset.csv` – Dataset used
- `requirements.txt` – Required Python libraries
Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

Outcome
Successfully built and evaluated a Multiple Linear Regression model to predict house prices.
